Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add [file]: adds a file as it looks now to your next commit between your local branch and cloud branch
  • git commit -m ['description']: commits staged content as a new commit to your branch
  • git clone [url]: retrieves a repository from a hosted location
  • git commands are listed in the document that is saved on your favorites
  • git branch -d: this is how you delete a branch. need to be in another branch before you can delete
